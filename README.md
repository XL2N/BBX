# BBX

Projeto web estatico baseado no wireframe da plataforma BBX.

## Organizacao

- `css/`: estilos do projeto
- `js/`: scripts de interacao
- `image/`: imagens, logo e fav icon
- `templates/`: paginas HTML secundarias

## Paginas

- Planos
- Gerenciamento
- Sobre Nos
- Perfil
- Contato

## Como abrir

Abra o arquivo `index.html` no navegador. As demais paginas estao ligadas pela barra lateral.
